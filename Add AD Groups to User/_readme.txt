This script will add specified AD groups to AD users. 

Details about the files:

removeusers.csv:
Add the samAccountname of users here. 

GroupAdder.ps1:
Actual script that adds groups to AD users

ErrorLog.txt:
Any errors will be appended to this txt file.


How to use this script:
-Edit your CSV and add the samAccountname's of the users needed
-Open the script
-Edit the credential variable
-Specify the groups
-run
